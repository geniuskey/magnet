# 회의봇 사용자 가이드

## 회의봇이란?

회의봇은 자연어로 회의 일정을 조율할 수 있는 AI 어시스턴트입니다.
참석자 검색, 일정 확인, 회의실 예약까지 대화만으로 해결할 수 있습니다.

## 시작하기

### 접속 방법
웹 브라우저에서 `https://meeting.company.com` 접속 후 사내 계정으로 로그인합니다.

### 기본 사용법
화면 하단 입력창에 원하는 내용을 자연어로 입력하세요.

## 주요 기능

### 1. 회의 예약하기

**예시:**
```
"세미나실 A 오전 10시 예약해줘"
"대회의실 B 2시부터 3시까지 예약"
"내일 오후에 김철수님과 1시간 회의 잡아줘"
"다음 주 월요일에 마케팅팀 전체 회의 가능한 시간 알려줘"
```

**빠른 예약 (한 문장):**
회의실과 시간을 한 번에 말하면 즉시 예약됩니다.
```
"세미나실 A 오전 10시 예약해줘"  → 바로 예약 완료
```

**단계별 예약:**
회의실을 먼저 선택한 후 시간만 말해도 됩니다.
```
1. (UI에서 소회의실 B 선택)
2. "오전 10시"  → 선택된 회의실에 예약 완료
```

**과정:**
1. 참석자 확인 (동명이인이 있으면 부서로 구분)
2. 빈 시간대 검색
3. 옵션 제시 (최대 5개)
4. 시간 선택
5. 회의실 추천
6. 최종 확인 후 예약

### 2. 일정 확인하기

**예시:**
```
"내일 내 일정 보여줘"
"다음 주 이영희님 일정 확인해줘"
"오늘 오후에 내가 시간 있어?"
```

> **참고:** 타인의 일정은 바쁨/가능 여부만 확인 가능합니다. 상세 내용은 본인만 볼 수 있습니다.

### 3. 회의실 찾기

**AI에게 요청:**
```
"화상회의 장비 있는 회의실 찾아줘"
"10명 수용 가능한 회의실 보여줘"
"5층에 있는 회의실 목록 알려줘"
```

**UI 필터 기능:**
- 이름 검색: 회의실 이름으로 검색
- 수용인원: 4인 이상, 8인 이상, 20인 이상 필터
- 장비: 프로젝터, 화이트보드, 화상회의, TV 필터
- 멀티플로어: 여러 층을 동시에 선택하여 비교

### 4. 회의 조건 지정

다양한 조건을 자연어로 지정할 수 있습니다:

- **시간대**: "오전에", "오후 3시쯤", "점심 이후에"
- **기간**: "이번 주", "다음 주", "월요일에"
- **회의 시간**: "30분", "1시간", "2시간 정도"
- **회의실 조건**: "프로젝터 있는 곳", "조용한 곳"

## 유용한 팁

### 빠른 적용 버튼
AI 응답에서 인식된 건물, 층, 회의실, 참석자, 시간이 버튼으로 표시됩니다.
- 버튼을 클릭하면 해당 항목이 즉시 적용됩니다
- 이미 적용된 버튼을 다시 클릭하면 선택이 해제됩니다 (토글)
- 여러 항목을 조합하여 빠르게 예약 조건을 설정할 수 있습니다

### 빠른 예약
시간, 참석자, 회의 시간을 한 번에 말하면 더 빠르게 예약할 수 있습니다.
```
"내일 오후 2시에 김철수, 이영희님과 1시간 회의 잡아줘"
```

### 문맥 인식 예약
회의실을 먼저 선택해 두면, 시간만 말해도 예약이 됩니다.
```
1. UI에서 "소회의실 B" 선택
2. "오전 10시" 입력
→ 소회의실 B에 설정된 회의 시간만큼 예약
```

### 옵션 선택
AI가 여러 옵션을 제시하면 번호나 구체적인 시간으로 선택하세요.
```
"1번으로 할게요"
"화요일 오전 10시로 해줘"
```

### 새 대화 시작
새로운 회의를 잡으려면 우측 상단의 "새 대화" 버튼을 누르세요.

## 인터페이스 기능

### 다크모드
- 우측 상단 테마 버튼으로 라이트/다크 모드 전환
- 시스템 설정에 따라 자동 적용 옵션 지원

### 키보드 단축키
- `←` / `→`: 이전/다음 날짜 이동
- `T`: 오늘 날짜로 이동
- `Esc`: 선택 취소

### 드래그 기능
- 타임라인에서 드래그하여 시간 범위 선택
- 내 예약을 드래그하여 다른 시간으로 이동
- 미니맵에서 드래그하여 전체 일정 빠르게 탐색

## 자주 묻는 질문

### Q: 동명이인이 있으면 어떻게 하나요?
A: AI가 부서 정보와 함께 선택지를 보여드립니다. 원하는 분을 선택해 주세요.

### Q: 예약 취소는 어떻게 하나요?
A: "회의실 A 2시 예약 취소해줘"와 같이 AI에게 요청하거나, 캘린더에서 직접 취소할 수 있습니다.

### Q: 외부 참석자도 추가할 수 있나요?
A: 현재는 사내 직원만 검색 가능합니다. 외부 참석자는 예약 후 캘린더에서 추가해 주세요.

### Q: 반복 회의도 예약할 수 있나요?
A: 네, "매주 반복", "매일 반복" 등으로 반복 회의를 설정할 수 있습니다. 종료 날짜도 지정 가능합니다.

### Q: 여러 층의 회의실을 동시에 보고 싶어요.
A: 층 선택 버튼에서 여러 층을 선택하면 (최대 4개) 동시에 볼 수 있습니다.

### Q: 빠른 적용 버튼을 취소하고 싶어요.
A: 이미 적용된 버튼을 다시 클릭하면 선택이 해제됩니다.
